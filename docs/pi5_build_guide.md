# Raspberry Pi 5 DustyCam Build Guide

## Hardware

- Raspberry Pi 5
- Raspberry Pi Camera (Module HQ recommended)
- microSD Card (16GB or larger)
- 3D Printed Case
- Battery (optional)
- UPS (optional)
- Solar Panel (optional)
- 3d printed case.

* Additional hardware can enable battery and solar support. See below for details.
